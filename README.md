# Serverless Data Lake on AWS

1. Prerequisites

```
brew install node
brew install hugo
```

2. Git clone

```
git clone https://github.com/nnthanh101/eks-workshop.git

cd eks-workshop
npm run theme
```

# Run Hugo

```
npm start
```

> http://localhost:8080

# Submodule

```
git submodule add https://github.com/nnthanh101/hugo-theme-learn themes/hugo-theme-learn
```
